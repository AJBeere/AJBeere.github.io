Kuri is an open-source Dead-Letter Queue-as-a-Service for distributed applications deployed on an AWS infrastructure.

Kuri integrates with an applications message queue and provides developers with a dashboard to allow interaction with failed messages. Kuri can also provide realtime Slack notifications of dead-lettered messages.
